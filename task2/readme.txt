Task 1 - Search for User Profiles
This project demonstrates how to search for user profiles using the profiles and emails tables from Task 1. Specifically, it provides an HTML, JavaScript, and PHP code for a single webpage that allows users to search for profiles by name.

Requirements
To run this project, you will need the following software installed on your system:

Web server software such as Apache or Nginx
PHP version 7.0 or higher
MySQL database server
Installation
To install this project, follow these steps:

Create a new database named web_task and import the profiles.sql and emails.sql files from Task 1.

Copy the contents of index.html, script.js, and search.php into files with the same names on your web server.

Modify the $servername, $username, $password, and $dbname variables in search.php to match your database connection details.

Access the index.html file in your web browser to start searching for profiles.

Usage
To use this project, follow these steps:

Enter a name into the search box on the webpage.

Click the "Search" button to run the search.

The search results, including the user's first name, last name, and email address, will be displayed on the page.
